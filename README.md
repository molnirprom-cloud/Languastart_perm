# LinguaStart

Адаптивный учебный сайт для русскоязычных пользователей: английский, итальянский и испанский языки.

## Установка

1. Установите Node.js LTS (рекомендуется 20+): https://nodejs.org
2. Откройте папку проекта:
   ```bash
   cd c:\Users\МSI\HTML_Practice\Site_languages
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```

## Запуск

```bash
npm run dev
```

Откройте в браузере `http://localhost:3000`.

## Структура проекта

```text
app/                    # страницы и роутинг Next.js (App Router)
components/             # переиспользуемые UI-компоненты и квизы
data/                   # данные языков, теории, материалов, квизов
hooks/                  # клиентские хуки (прогресс пользователя)
types/                  # типы TypeScript
utils/                  # локальное хранилище, форматирование, helper-функции
```

## Где менять данные

- Теория: `data/theory.ts`
- Слова и механики квизов: `data/quizzes.ts`
- Полезные материалы и ссылки: `data/resources.ts`
- Карточки языков: `data/languages.ts`

## Локальное хранение прогресса

- Ключ в `localStorage`: `linguastart.progress.v1`
- Логика чтения/записи: `utils/storage.ts`
- Кнопка сброса прогресса: вкладка `Мой прогресс` в разделе языка.
